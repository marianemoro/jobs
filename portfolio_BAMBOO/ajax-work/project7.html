Programa de Integração

Este informativo visa dar boas-vindas ao novos funcionários, apresentar algumas características da empresa e dar mais detalhes sobre os benefícios trabalhistas oferecidos.

O projeto foi executado em HTML5, utilizando animações e ícones para tornar o curso mais leve e atrativo. Para dividir melhor o conteúdo, foram criadas diferentes interações de clicar.

Como estratégia, durante o projeto foi possível trabalhar conceitos de User Experience, pois havia total liberdade para decidir que tipo de interação seria utilizada em cada etapa.

  • CLIENTE Calcenter
  • EMPRESA Ilog Tecnologia
  • CATEGORIA Design de Aprendizado
