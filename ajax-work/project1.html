Site Leroy Merlin

Desafio: o site da Leroy Merlin funciona como um lugar para encontrar informações sobre como fazer melhorias em casa, quais melhores soluções para cada tipo de necessidade, mas também é um e-commerce. O site foi contruído no modelo Ágil, passou por diversas melhorias, funcionalidades novas e está em um momento de reformulação das principais páginas: página de produto, página de lojas e a funcionalidade chamada de quicksearch, onde o usuário digita algum termos e o site retorna algumas sugestões de categorias, produtos e conteúdo relacionados.

Para o projeto da página das lojas, foram realizadas atividades com os stakeholders, para mapear o que estaria faltando na página, quais as prioridades de informações para organizar melhor os conteúdos e o lugar de algumas novas funcionalidades. Foram realizadas validações a partir dos protótipos de baixa fidelidade e, atualmente, está em fase de testes com usuários finais.

O projeto da quicksearch foi realizado a partir das principais pesquisas realizadas no site, tanto para termos buscados de produtos quanto de conteúdos, foi possível perceber que o site apresenta diferentes necessidades quando o usuário procura por um produto, um conteúdo específico ou uma loja. Portanto foram criados templates diferentes para cada necessidade.

O projeto da página das lojas irá substituir a atual, que apresenta diversos problemas de usablidade e coerência visual com o novo style guide. O projeto está em desenvolvimento, mas os protótipos foram todos construídos em conjunto com a área de SEO da empresa, para entender quais as necessidades e comportar os projetos da área de conteúdo e serviços da empresa, além de ajudar os clientes sobre como chegar na loja mais próxima, ou até mesmo fazer um tour virtual antes mesmo de sair de casa.
