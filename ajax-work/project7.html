Antônio dos Botos

O livro infantil, apresenta contos folclóricos ligados a regiões típicas açorianas de Florianópolis.

Como parte do projeto gráfico, foi chamada uma ilustradora para compor o livro, utilizando imagens de características bem marcantes para contextualizar visualmente esses contos.
