KP Outreach

Desafio: Criar o design de serviço de pré-atendimento através da interface on-line para o sistema de saúde norte americano.

Solução: Um sistema de pré-atendimento, onde o paciente é capaz de apontar os principais sintomas para o profissional de enfermagem no momento da triagem e, logo depois, ser encaminhado para um especialista realizar uma análise mais detalhada do caso. O sistema prevê envio de imagens, acesso a webcam e informações detalhadas dos profissionais responsáveis pelo atendimento.

Foi utilizado o processo de pesquisa exploratória como base para a construção do protótipo e o design de interface.

Diferencial: O sistema apresenta maior flexibilidade, melhora a frequência e consistência de atendimento, tanto para os profissionais de saúde quanto para os pacientes. Reduz a quantidade de consultas externas e riscos de reinternações.
