Glace App, manicure on demand

The app was a MVP to manicure and wax services, the users can find a professional near to them and schedule an appointment.

On this project:

Brand, styleguide, ui design, user flow, MVP

The challenges

The project has started as a wireframe from another designer, just to help the investors to understand the basic user flow. The challenge here was identfy the interface gaps and create a more complete and visual flow.

This project issues were more about branding and user interface than user experience because I didn't have the opportunity to test with the users or have an initial interview.

The process

At the beginning I draw a initial user flow to understand how the interface should work and what was been requested to be able to development.

The next step was design and validate the visual basic part.

The visual

After decide about branding and colors, the next step was design the user interface.

Learnings

An UI Design project that was already started.

Work in a started project can be very challenging, even if the job comes remotely from another country, you don't have an easy access to the user to test if your choices are good or talk with the team to have more ideas. The biggest learning here was about communication.
