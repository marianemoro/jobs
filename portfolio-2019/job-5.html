Glace App, manicure on demand

The app was a MVP to manicure and wax services, the users can find a professional near to them and schedule an appointment.

On this project:

Brand, styleguide, ui design, user flow, MVP

The challenges

The project has started as a wireframe with another designer, just to help the investors to understand the basic user flow. The challenge here was identfy the interface gaps and create a more complete flow.

This project was more about branding and user interface than user experience because I didn't have the opportunity to test with the users or have any inicial interview.

The surprises

Branding and styleguide project.

After the first fase, where I worked with the complete flow and validate with the manager, I started to create a brand and a color sheme to have a foundation for a styleguide.
