Mercado Sambaqui Branding

This project was to create a brand and a visual communication for a grocery store.

Mercado Sambaqui is a grocery store located in one of the oldest neighbourhoods in Florianopolis. The Azores culture is very strong, but in the last years the neighbourhood has been inhabited with new residents coming from large urban centres. The architecture is beautiful and attracts many tourists. So the branding proposal works with the 3 main public groups: old residents, new residents and tourists.

To develop the branding design and visual communication of the store it was necessary to start with the process of the immersion in the local culture to understand the 3 public groups of the store.
