PIA HTML Template

Documentation by Andy Lu

Thanks for buying "PIA", the premium website template. In this help documentation, I will walk you through the template files, this information may help you with customizing the template for your own use.

Installation

To install the HTML template, you need to upload the "pia" folder in your downloaded package to your host's root directory(or any other directory accesible). To make the contact form work please refer to section PHP Script, and be sure that your server can send email with the PHP "mail()" function.

HTML Structure

This template is based on Bulma, a free open source CSS framework based on Flexbox, to understand how to use bulma framework, please reference to the documentation.
