How to edit the Mockup?

You can edit this item in two ways: Using Mockupmator Lite or using the ‘old school’ method.

Using Mockupmator Lite

We built this tool to make your life easier. Mockupmator Lite is a Photoshop Extension that will help you automatize the customization process.

Download and enjoy it!

⚠️ Mockupmator Lite requires Photoshop CC 2018+, in case you can’t use this extension, we recommend that you use Photoshop Actions.

Get more help about Mockupmator Lite

Using the ‘Old School’ method.

  1. Open the Mockup in Photoshop.
  2. Edit the contents of the Smart Objects marked with Violet colors, They are located at the beginning of the document.
  3. Save changes after placing your own design.
  4. Go back to the mockup and check out the results.

How to customize the mockup.

  1. Select the layers marked with Yellow.
  2. Toggle hide to view the changes, or simply edit the opacity value or do any experiment with them.
