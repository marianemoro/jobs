Tatos App, fantasy sports

The app was a MVP to a fantasy sports app, the user could create theirs own teams to compete with other users and bet money. This kind of virtual game is very popular in United States.

On this project:

Brand, styleguide, ui design, user flow, MVP

The challenges

The project has started as a wireframe with another designer, just to help the investors to understand the basic user flow. For the first version, we decided to have one kind of sport, so we choose the football. Then I had to learn about this sport to design the whole solution.

After to understand how the sport works and have the basic user flow, I designed a brand, then I started to create a more complex interface and the style guide to create a high fidelity prototype.

This project was more about branding and user interface than user experience because I didn't have the opportunity to test with the users or have any inicial interview.

The surprises

The first branding and styleguide project

On this app I could create the brand, styleguide and mobile interface, and it was my first opportunity to work on this kind of project.
